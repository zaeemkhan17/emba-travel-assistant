# EMBA Travel Assistant

AI-powered travel assistant for Columbia University EMBA program.

## Features
- Automated flight price monitoring
- Email notifications with booking recommendations
- Price trend analysis
- Multi-source price comparison
- Preferred airline filtering

## Deployment
This app is designed to run on Render. See deployment guide for setup instructions.

## APIs Used
- Amadeus Flight Search API
- Google Flights (web scraping)
- Gmail SMTP for notifications
